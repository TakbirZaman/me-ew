# Me & U 💕

A cute, interactive multi-step love proposal page featuring Tom & Jerry.

## ✨ Features
- 4 flirty questions with smooth card transitions
- "No" button that runs away when you try to hover it
- Animated gradient background with floating hearts
- Spinning rainbow ring around each image
- Festival love emoji rain on the final screen
- Fully mobile friendly

## 🛠️ Built With
- HTML5 & CSS3 (animations, glassmorphism, gradients)
- Vanilla JavaScript (no frameworks)
- Google Fonts — Pacifico + Quicksand
